Illinois Criminal Justice Reform Commission Template
====================================================

Basic static site template using [grunt](http://gruntjs.com/) and [assemble](http://assemble.io/).

# Installation

```bash
download zip or clone into directory
cd (into directory)
npm install
bower install
```
